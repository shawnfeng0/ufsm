## [0.4.0] - 2026-01-01

[0.4.0]: https://github.com/shawnfeng0/ufsm/releases/tag/v0.4.0

### Added
- Initial release of ufsm - Ultra-lightweight Finite State Machine library
- Header-only C++17 hierarchical finite state machine (HFSM) implementation
- Support for nested/composite states with automatic drill-down and drill-up behavior
- Type-safe event handling and transitions via C++ templates and CRTP
- Event deferral mechanism for deferred event processing
- Flexible transition system supporting sibling and hierarchical transitions
- Custom actions and guards for transitions
- State entry/exit hooks
- Comprehensive unit tests
- Example implementations demonstrating library features
